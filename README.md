# MKR-Level-3-OOP

## МКР Рівень 3

Репозиторій містить виконання чотирьох завдань з об’єктно-орієнтованого програмування мовою C++.

У роботі реалізовано:
- абстрактний клас геометричних фігур;
- цифровий лічильник з обмеженим діапазоном;
- шаблонний клас для копіювання двовимірного динамічного масиву;
- функції з різною кількістю аргументів із поверненням результату через посилання та вказівник.

| [task-02-digital-counter](./task-02-digital-counter) | Завдання 2. Цифровий лічильник |
| [task-03-template-matrix](./task-03-template-matrix) | Завдання 3. Шаблонний клас копіювання двовимірного масиву |

## Завдання 1. Абстрактний клас геометричних фігур

У першому завданні реалізовано абстрактний клас `Figure` та похідні класи для різних геометричних фігур.

Реалізовано:
- абстрактний базовий клас;
- похідні класи фігур;
- віртуальні методи;
- переміщення, поворот, масштабування;
- зміну видимості;
- пізнє зв’язування через масив вказівників на базовий клас.

Матеріали завдання:

- [Код і скріншоти завдання 1](./task-01-geometric-figures)

## Завдання 2. Цифровий лічильник

У другому завданні реалізовано клас цифрового лічильника з мінімальним і максимальним значенням.

Реалізовано:
- встановлення діапазону;
- збільшення значення на 1;
- автоматичне скидання після максимуму;
- повернення поточного значення;
- графічний інтерфейс у вигляді цифрового таймера.

Матеріали завдання:

- [Код і скріншоти завдання 2](./task-02-digital-counter)

## Завдання 3. Шаблонний клас копіювання двовимірного масиву

У третьому завданні реалізовано шаблонний клас `DynamicMatrix<T>` для роботи з двовимірним динамічним масивом довільного типу.

Реалізовано:
- копіювання за рядками;
- копіювання за стовпцями;
- копіювання вибраного рядка;
- копіювання вибраного стовпця;
- копіювання головної діагоналі;
- потокове введення-виведення через оператори `>>` і `<<`.

Матеріали завдання:

- [Код і скріншоти завдання 3](./task-03-template-matrix)

## Завдання 4. Функції з різною кількістю аргументів

У четвертому завданні реалізовано функції, які дозволяють звернення з різною кількістю аргументів.

Реалізовано три підходи:
- перевантаження функцій;
- функції з параметрами за замовчуванням;
- функції зі змінною кількістю параметрів `f(int&, ...)`.

Також реалізовано повернення результату:
- через посилання `int&`;
- через вказівник `int*`.

Матеріали завдання:

- [Код і скріншоти завдання 4](./task-04-function-sum)
